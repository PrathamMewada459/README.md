# 🔴 OpsPulse — Quick Commerce Intelligence

**OpsPulse** is a real-time "Health Dashboard" designed for the high-stakes world of quick commerce. While standard dashboards just show you what is happening, OpsPulse calculates a **Business Stress Score** to show you how close you are to an operational meltdown.

Built for the "10-minute delivery" era, this platform turns raw data into a live command center for owners and operations managers.

---

## 🚀 The Core Concept: The Stress Score
The heart of this project is a weighted algorithm that monitors four critical pillars of a quick commerce business:

1.  **Inventory Health:** Real-time stock levels across categories.
2.  **Delivery Velocity:** Tracking SLA breaches against the "promise time."
3.  **Demand Pressure:** Analyzing orders-per-hour against total capacity.
4.  **Customer Friction:** Monitoring cancellation rates and support ticket spikes.

When the score drops, the UI adapts, culminating in a **War Room** mode for critical incidents.

---

## 🛠 Features

### 1. Adaptive Persona Views
The dashboard identifies who you are and changes the data architecture accordingly:
 **👔 Business Owner View:** Focused on "The Big Picture"—revenue trends, 14-point delivery averages, and cancellation radar charts.
 **⚙️ Operations Manager View:** Focused on "The Now"—live stock by category, active delivery partner deployment, and a prioritized "Action Item" queue.

### 2. Intelligent Incident Response
 **War Room Active:** If the Stress Score drops below 40, the system triggers a high-alert "Red State" with an emergency incident response checklist.
 **Live Ticker & Alerts:** A scrolling marquee for system-wide anomalies and a toast notification system for instant updates.

### 3. Scenario Engine (Demo Mode)
To showcase the dashboard's reactivity, we built a simulation engine. You can instantly toggle the business state between:
 **Normal Ops:** Healthy, green-state monitoring.
 **Delivery Meltdown:** Triggers SLA breaches and high stress.
 **Peak Opportunity:** High order volume with surge pricing indicators.

---

## 💻 Tech Stack
 **UI/UX:** HTML5, CSS3 (with custom variables for a dynamic theme engine).
 **Visualization:** `Chart.js` for real-time line, radar, and stacked bar charts.
 **Intelligence:** Pure Vanilla JavaScript for state management and the stress score algorithm.
 **Icons/Fonts:** IBM Plex Mono for that "industrial/data" feel and Plus Jakarta Sans for readability.
